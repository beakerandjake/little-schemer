# little-schemer

contains code from "The Little Schemer" book.

## Usage

I use tmux, vim and chez to run locally.

#### Start
```
bin/start.sh <file-name>
```

#### tmux commands

Kill pane
```
ctrl + b + x
```

Switch panes
```
ctrl + b + (up arrow / down arrow)
```

#### Chez reload file
```scheme
(load "./filename.scm")
```
